# Bluestock MF Capstone

Mutual Fund analytics capstone project — ETL, EDA, performance metrics, dashboard and advanced analytics.

## Structure
- `data/raw/` — original NAV JSON files from mfapi.in
- `data/processed/` — cleaned and merged CSVs
- `data/db/` — SQLite schema (db file excluded from git)
- `notebooks/` — Jupyter analysis notebooks
- `scripts/` — ETL pipeline, metrics, recommender
- `sql/` — schema.sql and queries.sql
- `dashboard/` — Power BI / Streamlit dashboard
- `reports/` — Final report and presentation
